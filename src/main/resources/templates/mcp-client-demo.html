Java Client 호출

호출 흐름

HTTP JSON-RPC
1 initialize context path가 반영된 MCP serverInfo와 tools capability를 확인합니다.
2 tools/list 현재 보호 객체에서 생성된 ORDS query tool 목록을 조회합니다.
3 tools/call Bearer Token이 입력된 경우 선택한 tool을 호출해 ORDS/VPD 결과를 받습니다.
Java MCP client 실행 결과가 여기에 표시됩니다.
