MCP SSE

보호 객체 ORDS 조회를 외부 MCP client에서 호출할 수 있도록 context path별 SSE transport로 제공합니다.

DB 연결 설정이 필요합니다.
DB 연결을 확인할 수 없습니다.

서비스 구성

기본 서비스 /mcp/sse 기존 MCP client 설정과 호환됩니다.
Context 서비스 /mcp/{contextPath}/sse 업무/환경별 이름으로 같은 ORDS tool set을 분리 노출합니다.
예시 /mcp/vpd-live/sse SSE 연결 시 message endpoint가 같은 context로 반환됩니다.

Endpoint

Default SSE /mcp/sse
Default Message /mcp/messages?sessionId={sessionId}
Context SSE /mcp/{contextPath}/sse
Context Message /mcp/{contextPath}/messages?sessionId={sessionId}
Context Path 영문/숫자로 시작하고 영문/숫자/_/-만 사용합니다. 예: vpd, ords-prod
Auth Basic Auth / BACKOFFICE_ADMIN_USER, BACKOFFICE_ADMIN_PASSWORD
Methods initialize, tools/list, tools/call

Tools

Name Object ORDS Path
ords.query.admin.board_posts ADMIN.BOARD_POSTS cb-ords/cb-object-query/admin/board_posts

{

  "method": "tools/list",

{

  "id": 3,
  "method": "tools/call",

      "limit": 50

  }
