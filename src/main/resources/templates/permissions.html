역할 선택

권한을 부여하거나 거부할 역할을 선택합니다.

선택한 역할의 민감도 허용 상한도 preview에 반영됩니다.

2

보호 객체 선택

등록된 보호 객체 또는 DB 스키마 객체를 선택합니다.

DB 스키마 객체를 선택하면 저장 시 보호 객체가 자동 등록됩니다.

권한 효과와 행 규칙

권한별 컬럼 마스킹

이 역할이 이 테이블/뷰를 조회할 때 원문 표시를 허용할 마스킹 컬럼을 선택합니다.

선택 가능한 마스킹 컬럼
보호 객체를 선택하면 마스킹 대상 컬럼이 표시됩니다.

객체의 기본 민감도/마스킹 정책은 ORDS 조회 Handler 대상에서 관리하고, 여기서는 역할 권한에 붙일 원문 표시 예외만 정합니다.
